Este programa recibe a través de redireccionamiento un input de x numeros (del 1 al 6) los cuales serán "encriptados" a través de un algoritmo DFS recursivo. Sin saber la estructura del árbol, no sería posible entender la conversión.

Creado por A01027541 Maximiliano Sapién y A01027446 Karen Morgado

5/21/21

ITESM CSF

Es necesario usar la carpeta proporcionada "cb"
Ejecución a través de redireccionamiento en MAC bash Ej: ./main < cb/cb1.txt > resultado.txt
Ejecución a través de redireccionamiento en WIN cmd Ej: main < cb/cb1.txt > resultado.txt
El resultado con su tiempo de ejecución y operaciones estará disponible en el archivo resultado.txt
